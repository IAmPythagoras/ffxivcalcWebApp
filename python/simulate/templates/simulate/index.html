This app is still in active developement and there could still be some issues. If you enconter any please let me know through the discord!:)

The simulator lets you simulate a fight with up to 8 players.

The rotation bis solver lets you attempt to find a BiS given a rotation and player.

Note that the results might not be optimal and it is always better to recheck the damage using another DPS metric (to compare to another bis). You can read more about the testing of this solver and the results it gave on the github repo.
