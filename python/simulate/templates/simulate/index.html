Hey! If you have this program know that it is still under work and this is a beta version! If you see or experience any problems please let me know through discord:)

The simulator lets you simulate a fight with up to 8 players and any number of actions.

The rotation bis solver lets you attempt to find a BiS given a fight layout. In other words, it attemps to find the bis given a rotation for a specific player.

Note that the results might not be optimal and it is always better to recheck the damage using another DPS metric (to compare to another bis). You can read more about the testing of this solver and the results it gave on the github repo.

If you require help you can click the help button above or join the discord and ask.
